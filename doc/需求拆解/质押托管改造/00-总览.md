# 质押托管改造总览

## 背景

本项目是 DApp 分销系统。`agent_daodun.docx` 中新增了质押托管、静态收益、服务费、AFI 加速、团队奖励、全球分红等业务。

本文档目录用于保存分批讨论后的需求结果，方便后续 Codex 或其他 AI 工具继续实现，不需要重新梳理上下文。

## 分批计划

1. `01-质押托管主流程.md`
   - 已讨论并确认。
   - 覆盖托管套餐、用户链上支付、后台拨付、订单状态、业绩加减、每日任务发放占位收益。
   - 后台拨付订单参与静态收益、业绩和升级计算，但不产生动态奖励。

2. `02-静态收益与G7.md`
   - 待继续细化。
   - 覆盖 G7 平均增长率、收益率区间、未推广用户特殊规则、指定钱包收益率。

3. `03-服务费与全球分红奖池.md`
   - 待继续细化。
   - 覆盖服务费内扣、平台沉淀、50% 进入全球分红奖池、后台调增调减。

4. `04-AFI质押加速.md`
   - 待继续细化。
   - 覆盖 AFI 质押、加速倍率、到期销毁、资产字段确认。

5. `05-团队奖励.md`
   - 待继续细化。
   - 覆盖直推、间推、极差、平级奖、有效托管订单条件。

6. `06-全球分红与后台管理.md`
   - 待继续细化。
   - 覆盖每周全球分红、节点等级加权、后台查询导出补发重算。

7. `07-托管等级重算.md`
   - 已讨论并确认。
   - 覆盖托管订单支付/拨付成功后的 Redis 异步等级重算、真实等级/赠送等级/管理员保底等级、个人业绩和小区业绩考核。

## 当前执行约定

- 先保存计划文档。
- 用户明确说“可以开始写质押托管主流程”后，再开始实现第一批代码。
- 第一批代码实现前，优先读取本目录所有文档和 `doc/表结构/xms_agent.sql`。
